Amélioration d'une Plateforme Web

Enrichissement fonctionnel d'une application existante

À propos du projet

Amélioration et enrichissement fonctionnel d'une plateforme web existante pour l'Atelier Numérique. Le projet consistait à étendre une application déjà bien structurée (architecture HMVC, Smarty, CodeIgniter) en y ajoutant de nouvelles fonctionnalités avancées et un système de gestion dynamique.

Fonctionnalités ajoutées :
  • Extension de l'architecture HMVC existante
  • Implémentation d'un page builder modulaire
  • Développement de vues SQL pour statistiques avancées
  • Amélioration de l'interface d'administration
  • Ajout de systèmes de reporting automatisé
Utilisateurs cibles :
Visiteurs

Recherche d'informations sur les services

Référents

Professionnels orientant les bénéficiaires

Administrateurs

Équipe gérant les parcours et le suivi

Technologies utilisées :
PHP CodeIgniter 4 Smarty MySQL MariaDB HTML5 CSS3 JavaScript Bootstrap Analyse Merise

  • Architecture HMVC

  • Gestion multi-utilisateurs
  • Interface d'administration

Développement par priorités

Priorité 1 – Gestion de contenu et page builder
  • Système de composition modulaire : Développement d'un page builder permettant l'assemblage de composants visuels standardisés
  • Dynamisation des contenus : Transformation des pages statiques en contenus éditables
  • Gestion des bénéficiaires : Amélioration avec prévention des doublons et validations
  • Fonctionnalités référents : Outils de gestion des ateliers et orientation
Priorité 2 – Outils statistiques avancés
  • Vues SQL optimisées : Création de vues complexes pour l'agrégation des données
  • Interface de visualisation : Dashboard interactif pour l'analyse des données bénéficiaires
  • Système de filtrage : Outils avancés pour le suivi personnalisé des parcours
  • Rapports automatisés : Génération automatique de statistiques (orientations, accompagnements, profils par structure)
Priorité 3 – Optimisation SEO
  • Optimisation technique : Amélioration du référencement pour une meilleure visibilité
  • Structure sémantique : Refactoring du code pour les standards web modernes
Approche de développement

Ce projet illustre une approche pragmatique d'amélioration et d'enrichissement d'une application existante déjà bien structurée. L'ajout de nouvelles fonctionnalités avancées (page builder, vues SQL, statistiques) sur une base technique solide démontre une maîtrise des enjeux d'évolution et d'extensibilité des applications web.

PHP Natif

Base de Données & Développement Backend

À propos du projet

Développement d'une application web complète en PHP natif avec une base de données MySQL. Ce projet illustre la maîtrise des concepts fondamentaux du développement web backend, de la conception de base de données avec l'analyse Merise jusqu'à l'implémentation complète.

Fonctionnalités principales :
  • Architecture MVC (Model-View-Controller)
  • Gestion complète des destinations (CRUD)
  • Gestion complète des utilisateurs (CRUD)
  • Système d'authentification sécurisé
  • Interface d'administration
  • Gestion de base de données relationnelle
Technologies utilisées :
PHP MySQL MariaDB HTML5 CSS3 JavaScript Bootstrap Analyse Merise
